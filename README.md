# Cow Anatomy SVG Components (React + Vite)

This project provides interactive cow anatomy SVG components as plain React components, with a Vite demo app.

## Components

- `src/components/CowHoofFootSvg.tsx`
- `src/components/CowHoofSideFootSvg.tsx`
- `src/components/CowLegSvg.tsx`
- `src/components/cowAnatomyIdMap.ts`

Each section is individually clickable and stylable.

## Run Locally

```bash
npm install
npm run dev
```

Open the shown local Vite URL (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Notes For Client Integration

You can copy the `src/components` folder into any React project and import the components directly.
